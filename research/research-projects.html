I am currently working on several research projects that explore how AI can be applied to diverse real-world problems, with direct links to source code and project documentation.

CoReason

AI-native learning platform for reasoning with LLMs, where students practice Framing, Judging, and Steering through iterative critique-and-improvement loops. Includes structured assessment modes, multilingual content, analytics, and PDF reporting for course workflows.

GitHub Repository

CoEval

Ensemble-based LLM evaluation framework where models rotate through teacher, student, and judge roles to build synthetic benchmarks, score responses, and generate interactive analysis reports. Includes multi-provider support, cost planning, and resumable experiment runs.

GitHub Repository

StableSteering

Interactive diffusion steering system for image generation that moves from a text prompt into preference-guided refinement loops. Ships with a FastAPI MVP, GPU-backed Diffusers runtime, session tracing, and practical docs for research and implementation workflows.

GitHub Repository
